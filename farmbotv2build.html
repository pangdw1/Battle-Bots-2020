Farmbot V2 Build

Our first project Farmbot takes on a new lease of life as Farmbot V2 

Hardware

After getting the go ahead from our head of project (Vaughn) to build a Battlebot,  we started dissembling 'farmbot' and started coming up with ideas to start our project.

Using existing materials from our previous project was a good starting point to get things rolling and being resourceful.

Alluminium extrusion (2020v) was used to start our chassis.  We all decided that if we were to do this properly we would need extra resources like machinery and other tools to get this chassis built. 

We began the process of gaining access to the engineering school's resources (EPICentre).
